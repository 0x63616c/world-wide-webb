# Archived docs

Point-in-time records of completed migrations (M3-M9, the domain flatten, the
k3s go-live). They are kept verbatim as a historical record and are **not**
current reference material.

They still mention `text-your-ex` and `amp`, two products that were removed from
the repo on 2026-07-18. Those references are accurate for the period the
documents describe and are deliberately left unedited - rewriting a dated record
to erase a product that did ship would make the record false.

For current deployment reference see `docs/deployment-design.md` and
`docs/k3s-migration/DESIGN.md`.
