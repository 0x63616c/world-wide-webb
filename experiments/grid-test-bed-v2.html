score weights

What the score weights do

The score ranks candidate layouts (lower = better). Each weight sets how much one quality matters. They only bite when candidates N > 1 (best-of-N keeps the lowest-scoring layout). 0 = ignore this entirely · 100 = make it dominate.
longest line
penalizes the single longest unbroken straight seam. The main knob for your "no long lines" goal. Try ~90-100.
same-size adj
penalizes neighbouring tiles being the same size. Raise it to force more size variety. ~40-60.
uniformity
penalizes every tile having a similar area. Raise it to reward a wide spread of big + small. ~30.
slivers
penalizes long thin tiles (aspect ≥ 2.5). Raise it for chunkier tiles. ~25, or 0 if you like slivers.
Example for your goal: longest line 100, same-size 50, uniformity 30, slivers 20 — then push candidates N up so it actually searches.
